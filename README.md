## ⚒️ Featured Project

### 🚀 [What's That In Space?](https://whatsthatin.space)

> An interactive space education experience featuring 3D planet exploration, real-time ISS tracking, and immersive visualisations of our solar system.

<p>
  <a href="https://whatsthatin.space"><img alt="Live site" src="https://img.shields.io/badge/Live-whatsthatin.space-F97316?style=for-the-badge&logo=vercel&logoColor=white" /></a>
  <a href="https://github.com/CodedVisionDesign/WhatsInSpace"><img alt="Source code" src="https://img.shields.io/badge/Source-GitHub-181717?style=for-the-badge&logo=github&logoColor=white" /></a>
</p>

**✨ Features**

- 🪐 **3D Solar System Explorer** — R3F planets, NASA-sourced textures, orbital data
- 🛰️ **ISS Tracker** — 3D model, live positional data, astronaut day-in-the-life timeline
- 🌕 **Artemis II Mission** — animated lunar trajectory, crew profiles, multi-layer Earth
- 🕳️ **Black Hole Visualisation** — event horizons, accretion discs, gravitational lensing
- ⚖️ **Planetary Weight Calculator** — your weight on every planet
- 📽️ **Voyager II Scroll Experience** — frame-by-frame scroll animation
- 🌙 **Moon Gallery** — major satellites with imagery and scientific data

**🛠 Build & Ops**

- Static export via `next export` for fast, cacheable delivery
- Git LFS for 3D GLB models · WebP frame sequences (~90% smaller than TIF)
- GitHub Actions CI/CD → FTPS deploy to Hostinger VPS
- Lighthouse CI gates (a11y 90 · perf 70 · SEO 80 · best-practices 80)
- British English enforced via cspell · ESLint · no em dashes policy
